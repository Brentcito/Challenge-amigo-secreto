# Challenge: Amigo Secreto

Este proyecto es una aplicación web que simula el juego del Amigo Secreto.  
Fue desarrollado utilizando HTML, CSS y JavaScript como parte de un challenge de programación.

## Características
- Permite registrar una lista de participantes.
- Asignación aleatoria sin que una persona se asigne a sí misma.
- Interfaz sencilla y fácil de usar.
- Funciona directamente en el navegador.

## Estructura del proyecto
- `index.html`: Página principal.
- `style.css`: Estilos de la aplicación.
- `app.js`: Lógica del juego.
- `assets/`: Carpeta con imágenes y recursos.

## Uso
1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en un navegador.
3. Ingresar los nombres de los participantes.
4. Presionar el botón para realizar el sorteo.

## Tecnologías
- HTML5
- CSS3
- JavaScript
